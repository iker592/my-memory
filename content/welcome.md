# Welcome to My Memory

This is your personal knowledge base and memory repository.

## Features

- **Markdown Support**: Write your thoughts in Markdown
- **Easy Navigation**: Browse through all your notes
- **Clean Interface**: Focus on what matters

## Getting Started

Start by creating new markdown files in the `content` directory. They will automatically appear in your file browser.

## Tips

- Use headings to organize your thoughts
- Add code blocks for technical notes
- Use lists for quick notes
